# User Personas

## Purpose

Keep feature decisions grounded in real users.

## Reference When

Use when prioritizing features or designing UX.

## AI Agents Must Obey

Optimize for the primary personas before adding advanced workflows.

## Personas

### Inference Engineer

Needs to understand latency regressions across model deployments. Cares about
TTFT, throughput, concurrency collapse, reproducibility, and CI gates.

### ML Platform Engineer

Needs repeatable benchmarks for internal endpoints. Cares about provider
compatibility, artifacts, automation, and clear failure modes.

### Application Engineer

Needs to know whether an LLM endpoint is fast enough for a feature. Cares about
simple commands, readable reports, and actionable recommendations.

### Open-Source Maintainer

Needs examples, clean architecture, and a contribution path. Cares about tests,
docs, and limited dependency sprawl.
