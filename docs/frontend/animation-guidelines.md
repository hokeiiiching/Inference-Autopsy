# Animation Guidelines

## Purpose

Define when motion is appropriate.

## Reference When

Use when adding report transitions, chart animation, or UI feedback.

## AI Agents Must Obey

Motion must support comprehension and never obscure metrics.

## Rules

- Prefer no animation for static reports.
- Use short transitions only for state changes.
- Respect reduced-motion preferences.
- Do not animate metric values in a way that delays reading.
- Do not use decorative motion.

## Acceptable Motion

- Subtle row highlight on hover or focus.
- Expand/collapse transitions under 150ms.
- Chart hover interactions.
