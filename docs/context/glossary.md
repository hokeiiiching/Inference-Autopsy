# Glossary

## Purpose

Provide canonical language for code, docs, CLI, and reports.

## Reference When

Use when naming fields, modules, docs, or user-facing strings.

## AI Agents Must Obey

Use these terms consistently.

## Terms

- Baseline: previous run used for comparison.
- Candidate: new run being compared against a baseline.
- Gate: rule that passes or fails a comparison.
- JSONL: newline-delimited JSON with one record per line.
- Profile: workload definition.
- Regression: metric change that violates a threshold or quality expectation.
- Replay: rerunning captured workload shape against an endpoint.
- Run: one benchmark execution.
- Static report: HTML artifact that opens without a server.
- Trace: request-level record including timings, metrics, status, and errors.
- Workload: set of prompts and settings used during a run.
